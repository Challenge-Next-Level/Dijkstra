⏰ 시간
- 매주 월요일 및 토요일 줌 미팅 ( 주 2회 )

💻 스터디 방식
- github을 통해 문제 풀이 및 주제 정리 공유
- 주중 (월 ~ 금)
    - 프로그래머스 코딩테스트 고득점 kit 문제 풀이 및 해당 주제 정리

- 주말 (토 ~ 일)
    - 주중 공부한 주제 관련 문제를 백준에서 선정 후 풀이
